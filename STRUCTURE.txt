AUTOMATION WORKFLOW - THE MAGIC!
================================
This is what you asked for: Automation!

1. Drop SVG files in input/
2. Run:  npm run convert-svg
3. Get JavaScript code in output/svg-shapes-to-add.js
4. Copy addSvgShape() calls into src/js/svg-shapes.js
5. Run:  npm run build
6. Open shape_wall.html

See your new shapes instantly!

QUICK COMMANDS
==============
npm run build              - Rebuild shape_wall.html from src/
npm run dev                - Build and auto-open in browser
npm run convert-svg        - Convert SVG files to addSvgShape() calls
