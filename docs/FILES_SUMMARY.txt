SOURCE CODE (For Developers)
============================

Build System:
  build.js                   - Merges src/ into shape_wall.html
  package.json               - NPM config

Source Folder (src/):
  index.html                 - HTML template
  css/style.css              - All styling
  js/shapes.js               - 8 built-in shapes
  js/svg-shapes.js           - Your custom SVG shapes go here
  js/canvas.js               - Canvas rendering
  js/ui.js                   - UI & colors
  js/utils.js                - Helper functions
  js/main.js                 - Initialization
