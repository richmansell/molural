HOW TO USE SVG SHAPES
=====================

1. Copy a shape from EXAMPLE_SHAPES.js
2. Paste into src/js/svg-shapes.js
3. Run: npm run build
4. Done! Shape appears in your app

OR

1. Draw a shape in Figma/Inkscape (100x100 canvas)
2. Export as SVG
3. Copy the SVG code into src/js/svg-shapes.js using addSvgShape()
4. Run: npm run build

QUICK API
=========

Add a shape:
  addSvgShape('key', 'Display Name', `<svg>...</svg>`);

Load from file:
  loadSvgShapeFromUrl('key', 'Display Name', '/path/to.svg');
